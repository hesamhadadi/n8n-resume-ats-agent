# n8n Resume ATS Agent

AI-powered job application agent using Gemini and Telegram notifications.

## Features
- 📄 PDF Resume Upload
- 🤖 Gemini AI Analysis
- 🎯 Job Matching
- 📊 ATS Score Calculation
- 📱 Telegram Notifications
- 💼 Automatic Applications

## Installation
Import the JSON workflow into n8n

## Environment Variables
- `GEMINI_API_KEY`
- `TELEGRAM_BOT_TOKEN`
